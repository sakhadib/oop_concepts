# A project to use general mathematical functions

## add method in java
```java
public int add(int a, int b){
  return a+b;
}
```
## subtract method in java
```java
public int subtract(int a, int b){
  return a-b;
}
```
## multiply method in java
```java
public int multiply(int a, int b){
  return a*b;
}
```
